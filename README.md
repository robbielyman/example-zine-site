# Example disco Zine Site

[See the documentation](https://disco.cloud/docs)

## To run locally

You'll need to grab a release of [zine](https://github.com/kristoff-it/zine).

```
  zine
```

The Dockerfile in the repository points to version 0.11.2 of Zine;
you'll want to update that line as part of upgrading to newer versions of Zine.
